# SDG 3: AI-Assisted Skin Cancer Classification

## Project Overview
This project develops a deep learning system for skin lesion classification using the HAM10000 dataset.

This work supports SDG 3: Good Health and Well-Being.

---

## Models
- MobileNetV2
- DenseNet121
- EfficientNetB3
- Ensemble

---

## Results
- Accuracy (Ensemble): 76.51%
- External Test (ISIC 2019): 78.92%

---

## Dataset
HAM10000 (Skin Cancer MNIST)

https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000

---
